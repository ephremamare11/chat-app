# 💬 Real-Time Chat App

A full-stack real-time chat application built using React, Node.js, Express, and Socket.io. Users can join with a username and send messages instantly without refreshing the page.

## 🚀 Live Demo
https://chat-app-c4qg.onrender.com

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Real-time Communication: Socket.io
- Deployment: Render

## ✨ Features
- Real-time messaging (no page refresh)
- Username-based chat
- Responsive UI
- Full-stack deployment

## 📂 Project Structure

```bash
chat-app/
│
├── client/ # React frontend
│ ├── src/

│ └── package.json
│
├── server.js # Backend server
├── package.json # Backend dependencies
└── README.md
```

## ⚙️ How It Works
1. User enters a username and sends a message
2. Frontend sends message using Socket.io
3. Server receives and broadcasts the message
4. All connected users see the message instantly
